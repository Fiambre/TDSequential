# tdsequential

**tdsequential** es una librería de Python que calcula el indicador técnico **TD Sequential** (incluyendo la fase de *Setup* y *Countdown*) según las reglas originales desarrolladas por Tom DeMark. Permite identificar posibles puntos de giro en la tendencia del mercado con base en secuencias específicas de barras alcistas o bajistas.

## Instalación

Puedes instalar esta librería desde PyPI usando pip:

```bash
pip install tdsequential
